ASCII Galaxian is a freeware program.
Works on Windows 7/8/8.1/10 
Created by: Shivam Shekhar

Description:
This is just a test version of my ASCII Galaxian game. 
The game is yet to be completed. 

Click on "Start.exe" to begin.

Controls:
A: move Left
D: move right
W: fires primary missile
P: fires dual lasers
[: fires left laser
]: fires right laser
L: saves game

Game Play:
-Destroy all the incoming enemy ships.
-When your health falls to 0, or when the enemy ship strikes you, 
 you lose.
-They enemies have varying health, with the blue being the
 strongest and the red being the weakest.
